What is production-shaped

  • Receiver-held policy gates the operation.
  • Caller proof, wallet proof, issuer credentials, and Dregg-shaped evidence are checked as separate layers.
  • Resource locks can allow exact matches and reject mismatches.
  • Verify/execute receipts are inspectable and redaction-safe.

What is not proven

  • There is no live Castalia/Dregg network call in this browser demo.
  • No production deployment proof.
  • No public chain anchoring or immutable publication.
  • No runtime Dregg blocklace finality / BLS threshold QC verification.
  • No Midnight proof verification or Cardano settlement finality.
